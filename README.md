# 한랭 (HanLang) 🇰🇷

**한글로 코딩하는 프로그래밍 언어**

한랭(HanLang)은 한글 키워드를 사용하는 프로그래밍 언어입니다. Python으로 작성된 인터프리터와 VSCode 스타일의 IDE를 제공합니다.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## 실행 방법

```bash
# IDE 실행
python hanlang_ide.py

# 또는
python run_ide.py
```

## 주요 기능

### 언어 기능
- 변수/상수 선언
- 함수 및 클래스
- 조건문 (`만약`/`아니면만약`/`아니면`)
- 반복문 (`반복`/`동안`)
- 람다 함수 (`(x) => x * x`)
- 삼항 연산자 (`조건 ? 참값 : 거짓값`)
- 딕셔너리/맵 (`{"키": "값"}`)
- 예외 처리 (`시도`/`잡기`/`마침내`/`던지기`)
- 50개 이상의 내장 함수

### IDE 기능
- 구문 강조
- 자동완성 (Ctrl+Space)
- 찾기/바꾸기 (Ctrl+F, Ctrl+H)
- 코드 접기/펼치기
- 터미널 입력 지원
- 줄 번호 표시

## 문법 예제

### 변수 선언
```
변수 이름 = "홍길동"
변수 나이 = 25
상수 파이 = 3.14159
```

### 조건문
```
만약 나이 >= 20 {
    출력("성인입니다")
} 아니면만약 나이 >= 14 {
    출력("청소년입니다")
} 아니면 {
    출력("어린이입니다")
}
```

### 반복문
```
반복 i = 1 : 10 {
    출력(i)
}

동안 조건 {
    출력("반복 중...")
}
```

### 함수
```
함수 더하기(a, b) {
    반환 a + b
}

변수 결과 = 더하기(3, 5)
출력(결과)  # 8
```

### 람다 함수
```
변수 제곱 = (x) => x * x
출력(제곱(5))  # 25

변수 더하기 = (a, b) => a + b
출력(더하기(3, 4))  # 7
```

### 삼항 연산자
```
변수 상태 = 나이 >= 18 ? "성인" : "미성년자"
출력(상태)
```

### 클래스
```
클래스 사람 {
    함수 생성(이름, 나이) {
        나.이름 = 이름
        나.나이 = 나이
    }

    함수 소개() {
        출력("저는", 나.이름, "이고,", 나.나이, "살입니다.")
    }
}

변수 학생 = 사람("김철수", 20)
학생.소개()
```

### 딕셔너리
```
변수 사람 = {"이름": "홍길동", "나이": 30}
출력(사람["이름"])  # 홍길동

사람["직업"] = "개발자"
출력(키값들(사람))  # ["이름", "나이", "직업"]
```

### 예외 처리
```
시도 {
    던지기 "오류 발생!"
} 잡기 (오류) {
    출력("오류:", 오류)
} 마침내 {
    출력("항상 실행됨")
}
```

### 리스트
```
변수 과일들 = ["사과", "바나나", "오렌지"]
추가(과일들, "포도")
출력(과일들)  # ["사과", "바나나", "오렌지", "포도"]
출력(길이(과일들))  # 4
```

## 내장 함수

### 기본 함수
| 함수 | 설명 |
|------|------|
| `출력(값, ...)` | 화면에 출력 |
| `입력(프롬프트)` | 사용자 입력 |
| `길이(대상)` | 길이 반환 |
| `정수변환(값)` | 정수로 변환 |
| `실수변환(값)` | 실수로 변환 |
| `문자열변환(값)` | 문자열로 변환 |
| `타입(값)` | 자료형 반환 |
| `범위(시작, 끝)` | 범위 리스트 |

### 수학 함수
| 함수 | 설명 |
|------|------|
| `제곱근(x)` | 제곱근 |
| `거듭제곱(x, n)` | 거듭제곱 |
| `올림(x)` | 올림 |
| `내림(x)` | 내림 |
| `반올림(x, n)` | 반올림 |
| `사인(x)`, `코사인(x)`, `탄젠트(x)` | 삼각함수 |
| `로그(x)`, `로그10(x)` | 로그 |
| `파이`, `자연상수` | 상수 |
| `랜덤()` | 0~1 난수 |
| `랜덤정수(a, b)` | 정수 난수 |

### 문자열 함수
| 함수 | 설명 |
|------|------|
| `대문자(s)` | 대문자로 변환 |
| `소문자(s)` | 소문자로 변환 |
| `분리(s, 구분자)` | 문자열 분리 |
| `결합(리스트, 구분자)` | 문자열 결합 |
| `교체(s, 찾기, 바꾸기)` | 문자열 교체 |
| `공백제거(s)` | 양쪽 공백 제거 |
| `찾기(s, 검색어)` | 위치 반환 |
| `자르기(s, 시작, 끝)` | 부분 문자열 |

### 리스트/딕셔너리 함수
| 함수 | 설명 |
|------|------|
| `추가(리스트, 값)` | 리스트에 추가 |
| `제거(리스트, 값)` | 리스트에서 제거 |
| `삽입(리스트, i, 값)` | 위치에 삽입 |
| `정렬(리스트)` | 정렬 |
| `뒤집기(대상)` | 뒤집기 |
| `키값들(딕셔너리)` | 키 목록 |
| `값들(딕셔너리)` | 값 목록 |
| `항목들(딕셔너리)` | (키, 값) 목록 |
